Recent Work

Predicting Market Sentiment Using Gov Contracts

An ML model for assisting with Option Trading using publicly available federal government contract data.

Learn More

This project analyzes over 3 million rows of U.S. federal contract data to model market movement following key awards. It uses Python, Pandas, and Scikit-learn to classify and cluster market behavior, producing trade signals optimized for options trading. Tableau is used for exploratory visuals and summarizing high-yield contract clusters.

Automated Purchase Ordering

An automation process for spuring purchasing orders by tapping into the point of sale.

Learn More

This project streamlines the purchase ordering process by integrating real-time point-of-sale (POS) activity with backend inventory rules. I designed a system that detects when a threshold is met (e.g., low stock or high turnover) and automatically emails a purchasing order form for review or approval.
