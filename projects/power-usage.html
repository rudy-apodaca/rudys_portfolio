Power Usage & Weather Trends – Data Engineering Project

This project explores the relationship between weather conditions and energy consumption across residential and industrial sectors in the Pacific Northwest. The goal was to uncover how temperature spikes correlate with power demand and whether usage patterns differ significantly between consumer types.

Objectives:

  • Compare hourly power usage trends across residential and industrial sectors
  • Integrate weather variables (e.g., temperature) to detect correlations
  • Build a time-series dashboard for live or near-real-time data monitoring
  • Evaluate sector responsiveness to temperature anomalies using statistical methods

Stack:

Python, PostgreSQL, EIA API, NOAA API, Grafana, Pandas, SQL

Status:

Active analysis phase – querying real-time data sources and building time-series models with weather overlays
