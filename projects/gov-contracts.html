Predicting Market Sentiment Using Government Contracts

This project uses federal government contract award data to help predict short-term stock price movements. I built an ML pipeline that clusters stocks based on contract activity, then models post-award movement ranges to assist in directional options trading.

Key Features:

  • Automated data pull from USASpending + yFinance
  • K-Means clustering on stock movement characteristics
  • Implied volatility and anomaly detection integration
  • Outcome visualization with Seaborn + Matplotlib

Stack:

Python, Pandas, Scikit-learn, yFinance, Git, matplotlib, seaborn
